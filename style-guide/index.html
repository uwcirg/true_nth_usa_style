Tooltips

Example code for tooltips:
<span class="show-tooltip" data-toggle="tooltip" data-placement="left" title="Tooltip title"> blabla </span>

NOTE:
  • The show-tooltip class is MANDATORY
  • The data-toggle="tooltip" attribute is MANDATORY
  • The title attribute is MANDATORY (this attribute will be displayed as the content of the tooltip)
  • The data-placement="left" attribute is optional (you can choose among "right", "left", "top", "bottom")

Popovers:

Example code for popovers:
<span class="show-popover" data-toggle="popover" data-placement="left" data-content="Popovers content"> blabla </span>

NOTE:
  • The show-popover class is MANDATORY
  • The data-toggle="popover" attribute is MANDATORY
  • The data-content attribute is MANDATORY (this attribute will be displayed as the content of the popover)
  • The data-placement="left" attribute is optional (you can choose among "right", "left", "top", "bottom")
  • The title attribute is optional (this attribute sets the title of the popover)

Special symbols/icons:

Example code:
<span class="mo-question-empty"></span>
<span class="mo-question-full"></span>

You can also assign popover or tooltips to the icons. For example:
<span class="mo-question-empty show-tooltip" data-toggle="tooltip" title="Need help?"></span>
<span class="mo-question-full show-popover" data-toggle="popover" data-content="Need help?"></span>

Responsive Images:

<picture>

The "picture" tag will be used for "art directed" uploads. This means that there will be DIFFERENT images for different sizes of the picture. Below is an example with 4 different versions of an image. 3 different images (the 2 smaller ones cropped from the original large one then saved as separate files) plus one fallback-image (completely different again)

    <picture>
        <source media="(max-width: 450px)" srcset="files/ResponsiveSampleImage_smallCrop.png"/>
        <source media="(max-width: 700px)" srcset="files/ResponsiveSampleImage_mediumCrop.png"/>
        <source srcset="files/ResponsiveSampleImage_large.png"/>
        <img src="files/moustache-guy.jpg" alt="Fallback picture"/>
    </picture>

srcset

The "srcset" will be used for "normal" uploads. This means that there will be different SIZES of the SAME image.

Code used:

<img src="files/ResponsiveSampleImage-2_medium.jpg" srcset="files/ResponsiveSampleImage-2_small.jpg 450w, files/ResponsiveSampleImage-2_medium.jpg 700w, files/ResponsiveSampleImage-2_large.jpg 1024w" alt=""/>

Example:

Important notes:

  • 450w, 700w and 1024w are referred to the size OF THE PICTURE, NOT the size of the device
  • If you try to resize the browser to test which image is served, it might NOT work as you expected, because the browser, if using srcset, will ALWAYS serve the largest image it has on cache. Therefore if you want to test the serving differences, you've gotta open this page on incognito mode (so there is no cache) and the initial width of the browser must be narrow. Then expand the browser size and you'll see that the browser will change the size in bigger and it will keep serving that even if you resize down.
